This project uses React and Open Weather API to allow users to search a city's weather, display a city's time as well as whether it is day or night time.

Challenge - Time zone - My plan is when users type in the city, this will display local time, and show whether it is daytime or nighttime.

How to solve - Time zone - Create new date, convert to UTC, convert to milliseconds, add to epoch time to get local time. Compare with sunrise and sunset time.
