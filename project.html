My Work.

HTML + CSS - Responsive Website.

This website has a dynamic layout featuring several columns on desktop devices, which are displayed on a single column on handheld devices. Depending on whether a tablet or mobile device is used, the menu shrinks to a menu icon.

CSS + JavaScript - Animation

This is a lovely project combining CSS animation and JavaScript that allows users to choose which colour they will use to design the flowers on a PhotoShop interface.

JavaScript - To-Do List

This project idea came from my weekly to-do list whiteboard that I use to list everyday activities. The application allows users to add activities on a chosen day, mark the activities as done, and delete activities.

XMl + PHP - Ticket System

In this project, I implemented the support ticket system for which I designed XML data structures. I built a ticket system that has a login form, reads tickets' info, and sends messages for both users and admin.

PHP - E-Commerce

This E-commerce website allows shoppers to purchase digital products and pay online. It also allows the admin to manage the website. Both users and admin can do CRUD.

PHP + JavaScript - API

This project combines GIPHY API and Twilio API. Users are able to search gifs and send an SMS to their cellphone.

C# - Passion Project

This project uses the asp.net framework to build a web application to manage one-to-many and many-to-many relationships between tables and allows users and admin to do CRUD.

React - Weather App

This project uses React and Open Weather API to allow users to search a city's weather, display a city's time as well as whether it is day or night time.
